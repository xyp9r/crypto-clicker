# 💰 Crypto Clicker

A lightweight, browser-based economic simulation game built with pure JavaScript. Experience the thrill of scaling a crypto-mining empire from scratch. Now fully optimized for mobile devices!

## 🎮 Gameplay Mechanics
1. **Click & Earn:** Start by manually clicking the coin to build your initial capital.
2. **Upgrade Your Setup:** Visit the shop to purchase GPUs once you have enough funds.
3. **Passive Income:** Each GPU generates automatic income every second.
4. **Dynamic Economy:** Strategic planning is required as hardware prices increase by **1.5x** after every purchase (inflation model).
5. **Random Events:** The market is unpredictable! Watch out for news banners that can multiply your income or crash the market.

## ⚙️ Tech Stack & Features
* **HTML5** — Semantic structure.
* **CSS3** — Custom styling, flexbox layouts, UI state management, and **Mobile Responsive Design**.
* **JavaScript (ES6+)** — Core game logic:
    * **Event Listeners:** Handling user interactions.
    * **Game Loop:** `setInterval` implementation for passive income ticks.
    * **DOM Manipulation:** Real-time UI updates for score, prices, and income.
    * **Local Storage:** Your progress is automatically saved. Close the browser and come back anytime!
    * **Visual & Audio Effects:** Particle animations on clicks and custom sound effects for in-game actions.

## 🚀 How to Run
This project is built with Vanilla JS and requires no build tools or dependencies.
1. Clone the repository.
2. Open `index.html` in any modern web browser (Desktop or Mobile).
3. Start clicking!
